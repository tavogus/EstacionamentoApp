# EstacionamentoApp

Atualização,modernização e melhoria do projeto estacionamento feito na faculdade como trabalho.


### Funcionalidades ###

Funcionalidades implementadas até agora:

- **Usuarios**
  - Cadastro
  - Pesquisa
  - Ativação/Desativação

- **Tarifas**
  - Cadastro 
  - Pesquisa
  
- **Veiculos**
  - Cadastro
  - Pesquisa

- **Movimentação**
  - Cadastro
  - Pesquisa
  - Fechamento
  

### Tecnologias antes usadas ###
- Servelt
- Jsp

### Tecnologias agora usadas ###
- Spring boot
- Spring data jpa
- Spring security
- Thymeleaf
- bootstrap
